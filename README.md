# App aniversari Franra

He volgut fer aquesta web interactiva per celebrar 3 anys amb la meva parella. He afegit unes quantes categories i coses que ens interessen a nosaltres. 

## Característiques

- **Introducció:** Una petita introducció de per què he fet aquesta app i què hi podem trobar. 
- **Cronografia:** Una cronografia de moments importants a la nostra relació.
- **Galeria:** Algunes fotos de paisatges que hem anat fent. 
- **Memes:** Una pàgina per veure memes random per fer-ho més divertit.

## Disseny

- Paleta de colors a partir de colors que ens agraden.
- Estil senzill però divertit.
- Responsive per mòbil i ordinador. 

## Tecnologies

- **React**
- **MUI Timeline** – Per fer el timeline.
- **Tailwind / CSS** – Per estil i per responsive.
